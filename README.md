# 1 项目介绍
基于SSM+Vue的新生宿舍管理系统：后端 SpringBoot、Mybatis-Plus，前端Vue+ElementUI，具体功能如下：
## 管理员
- 个人中心：查看和修改个人信息、修改密码
- 公告信息管理
- 院系管理
- 班级管理
- 学生管理
- 宿舍信息管理
- 宿舍安排管理
- 卫生检查管理
- 报修信息管理
- 报修处理管理
- 缴费信息管理

## 学生
- 个人中心：查看和修改个人信息、修改密码
- 公告信息管理
- 宿舍安排管理
- 卫生检查管理
- 报修信息管理
- 报修处理管理
- 缴费信息管理
